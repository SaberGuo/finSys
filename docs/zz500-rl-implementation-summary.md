# ZZ500 RL选股交易系统 - 实施总结

## 项目概述

成功实现了基于强化学习（RL）的中证500（ZZ500）股票选股和交易系统。系统使用单股票RL模型对343只ZZ500成分股进行独立评分，维护5只股票的投资组合，并实施智能风险管理。

## 核心创新

### 单股票RL架构

**传统方案的问题**：
- 多股票环境：观察空间 = 1 + (2+7) × 343 = 3088维
- 训练慢，收敛困难，计算资源需求高

**我们的方案**：
- 单股票模型：观察空间 = 1 + (2+7) × 1 = 10维
- 训练快速（20-30分钟），易于收敛
- 可扩展到任意数量股票，无需重新训练

## 系统工作流程

### 训练阶段
```
1. 加载343只ZZ500股票列表
2. 选择数据最完整的股票（000001.SZ）
3. 训练单股票RL模型（PPO，200K步）
4. 保存模型和元数据
```

### 推理阶段（每日）
```
1. 对343只股票逐一评分
   ├─ 构建单股票观察（10维）
   ├─ RL模型预测action
   └─ Sigmoid映射到评分[0,1]

2. 筛选评分>0.9的股票

3. 选择前5名

4. 按评分加权分配资金
   例：评分[0.95, 0.92, 0.91, 0.90, 0.88]
   权重：[20.8%, 20.2%, 20.0%, 19.7%, 19.3%]

5. 执行交易并监控风险
   ├─ 止损：-5%
   ├─ 止盈：+20%
   └─ RL退出：评分<0.8
```

## 技术亮点

### 1. Action到Score的映射

使用Sigmoid函数将RL的连续action映射到[0,1]评分：

```python
score = 1 / (1 + exp(-action))
```

**映射关系**：
- action > 2.2 → score > 0.9（强买入信号）
- action = 0 → score = 0.5（中性）
- action < -2.2 → score < 0.1（强卖出信号）

### 2. 评分加权持仓

根据RL评分动态分配资金，高评分股票获得更多资金：

```python
weight_i = score_i / sum(top5_scores)
allocation_i = total_capital * weight_i
shares_i = int(allocation_i / price_i)
```

### 3. 灵活的风险管理

- **固定止损止盈**：每只股票独立设置-5%止损和+20%止盈
- **动态RL退出**：当RL评分降至0.8以下时主动退出
- **阈值筛选**：只买入评分>0.9的高质量股票

## 使用方法

### 训练模型
```bash
python scripts/train_zz500_single_stock.py \
    --config config/zz500_rl_single_stock.yaml \
    --db-path data/processed/zz500_data.db \
    --output-dir models/zz500_single_stock
```

### 回测策略
```bash
python scripts/backtest_zz500_single_stock.py \
    --model models/zz500_single_stock/ppo_zz500_single_20240630_HASH.zip \
    --config config/zz500_rl_single_stock.yaml \
    --start 2024-07-01 \
    --end 2024-12-31 \
    --output-dir runs/zz500_backtest
```

## 配置参数

关键参数可在 `config/zz500_rl_single_stock.yaml` 中调整：

```yaml
# 训练参数
training:
  algorithm: "ppo"           # PPO/SAC/TD3
  total_timesteps: 200000    # 训练步数

# 选股参数
zz500_selection:
  portfolio_size: 5          # 持仓股票数
  score_threshold: 0.9       # 评分阈值
  score_mapping: "sigmoid"   # Action映射方法
  position_sizing: "score_weighted"  # 仓位分配方式
  
  # 风险管理
  stop_loss_pct: -0.05      # 止损-5%
  take_profit_pct: 0.20     # 止盈+20%
  rl_exit_threshold: -0.2   # RL退出阈值
```
